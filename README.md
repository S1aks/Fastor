# Fastor
Fas(t) (transla)tor Eng-Rus

Fastor - Fas(t) (transla)tor. Быстрый переводчик, когда нужно быстро найти перевод нужного слова (как с английского на русский, так и наоборот). При наборе предлагает варианты. Есть история слов.
В программе используется свободный API сайта https://skyeng.ru

Ссылка в Google Play: https://play.google.com/store/apps/details?id=com.s1aks.fastor

Ссылка в RuStore: https://apps.rustore.ru/app/com.s1aks.fastor
